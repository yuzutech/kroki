# Diagram as Code 0.1.0

Initial MVP release of the self-hosted Diagram as Code toolchain.

## Included

- Fastify rendering Gateway with scoped API-key verifier records, per-principal rate limit/cache isolation, bounded concurrency and queue, timeout, output-size and media validation, SVG sanitization, structured redacted logs, aggregate metrics, in-memory LRU cache, and single-flight request coalescing.
- Kroki 0.31.1 rendering for Mermaid, PlantUML, Graphviz/DOT, and D2 with canonical SVG output.
- VS Code extension with live preview, debounced render-on-save, SecretStorage API keys, explicit stable SVG export, and labeled status-bar/editor/preview/context-menu controls.
- GitHub Action that detects missing, stale, and orphaned committed SVG files without committing or commenting automatically.
- Docker Compose deployment files with an externally reachable Gateway edge network and an internal-only Kroki/Mermaid rendering network.

## Requirements

- VS Code 1.100 or later for the extension.
- Docker or Podman-compatible infrastructure for the rendering stack.
- A self-hosted GitHub runner when the Gateway is available only on a private network.
